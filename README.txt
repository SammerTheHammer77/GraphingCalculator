In order to use the function features, you must click the "f(x)" button once to put the
calculator into function mode. Afterward, press the "f(x)" button again to put an "X"
into the input string.

The graph is scalable if you scroll your mouse to zoom in and out.

There is keyboard input where certain buttons on the keyboard correspond with 
buttons on the calculator if you prefer to use the keyboard for input instead of the mouse.
